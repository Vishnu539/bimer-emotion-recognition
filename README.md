# BiMER – Bimodal Emotion Recognition

This project implements a bimodal emotion recognition system using speech and text modalities based on the BiMER architecture.

## Features
- Speech Emotion Recognition using Wav2Vec 2.0
- Text Emotion Recognition using BERT
- Intermediate Fusion for bimodal prediction
- Data augmentation for speech and text
- Evaluation on IEMOCAP dataset

## Dataset
IEMOCAP dataset was used for training and evaluation.

Dataset is not included in the repository due to licensing.

## Models
Pretrained models included:
- SER model
- TER model

## Installation

pip install -r requirements.txt

## Usage

Run the notebook:

notebooks/bimer_final.ipynb

Accuracy: ~88%
